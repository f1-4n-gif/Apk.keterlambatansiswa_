Mode Offline Aktif

Aplikasi ini dirancang untuk dapat beroperasi 100% tanpa internet. Anda dapat melakukan Scan Barcode menggunakan Scanner USB dan mencetak laporan secara normal. Pastikan Anda melakukan Backup Data secara berkala untuk menjaga keamanan data Anda.

1. Backup & Restore Data

  • Gunakan tombol Backup Data di menu sebelah kiri bawah untuk menyimpan seluruh data absensi dan daftar siswa ke dalam satu file.
  • Jika Anda pindah laptop atau tidak sengaja membersihkan riwayat Browser, gunakan Restore Data dan masukkan file backup yang pernah Anda unduh.

2. Mengelola Data Siswa

  • Pastikan Anda mengisi data siswa terlebih dahulu sebelum menggunakan fitur scan.
  • Gunakan fitur Input Massal untuk memasukkan banyak data sekaligus dari Microsoft Excel.
  • Unduh Format CSV, isi data (NISN, Nama Siswa, Kelas), simpan kembali sebagai CSV, lalu unggah.

3. Pindai NISN (Scan Barcode)

  • Fitur ini mencatat kehadiran secara otomatis.
  • Menggunakan Scanner USB: Pastikan kursor berkedip di dalam kotak input putih, lalu tembak barcode. Ini dapat bekerja 100% tanpa jaringan internet.
  • Menggunakan Kamera: Anda membutuhkan internet untuk memuat kamera pertama kali. Setelah dimuat, ia bisa digunakan terus.

4. Rekapan & Cetak Laporan

  • Rekapan Harian: Melihat siswa yang terlambat hari ini.
  • Rekapan Bulanan & Tindak Lanjut: Siswa yang terlambat 4 kali atau lebih akan ditandai merah (SP 1 / Panggilan Orang Tua).
  • Klik tombol Cetak PDF untuk mencetak laporan resmi (otomatis dilengkapi kop surat sekolah dan tanda tangan guru piket aktif).
